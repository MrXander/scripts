## setup_local_k8s_tools.sh
Prepares your local env for k8s. Scripts installs:
1. kubectl
2. kubectl bash completion
3. helm
4. helm bash completion
5. helm plugins: helm-github, helm-tiller, helm-restore
6. kubectx & completion
7. kubens & completion
8. kubectl aliases
9 kubetail
